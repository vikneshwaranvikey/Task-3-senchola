About Front End Developer

A front-end web developer it is the responsible for implementing visual elements that users see and interact with in a web application. they are usually supported by back-end web developers.

A Front-End Developer it is responsible for developing new user-feature, determining the structure and design of web pages, bulding reusable codes, optimizing page loading times, and using a variety of markup language to create the web pages.

Front End Language

The main-End language used in web development

HTML,CSS and JavaScript

HTML

The Hyper text markup language or HTML is the standard markup language for documents designed to be displayed in a web browser.

CSS

CSS stand for Cascading Style Sheets Css describes how HTML elements are to be displaye on screen.

JAVA SCRIPT

JavaScript is a dynamic programming language that's use for web development,in web application, for game development,and lots more.

BOOTSTRAP

Bootstrap is a free open source front-end development framework for the creation of websites and web apps.

REACT

React is a free and open-source front-end Javascript library for building user interfaces based on components.

ANGULAR

Angular is a platform and framework for building single-page client applications using HTML and TypeScript.
